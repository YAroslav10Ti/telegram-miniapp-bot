# Telegram Bot + Mini App

Учебный проект Telegram-бота с Mini App (Web App внутри Telegram).

##  Возможности

- Telegram бот (Node.js + Express)
- Webhook обработка сообщений
- Echo бот (повторяет сообщения)
- Telegram Mini App
- Получение данных пользователя
- UI внутри Telegram

## Технологии

- Node.js
- Express
- Axios
- Telegram Bot API
- Telegram WebApp API
- HTML / CSS / JS
